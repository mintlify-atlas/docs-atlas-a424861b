This page walks you through a complete working program that demonstrates both one-shot and streaming hashing.

## Complete example

```go main.go
package main

import (
	"fmt"

	"github.com/cespare/xxhash/v2"
)

func main() {
	// --- One-shot hashing ---

	// Hash a string directly (no []byte conversion, no allocation).
	h1 := xxhash.Sum64String("")
	fmt.Printf("Sum64String(\"\")    = 0x%x\n", h1)
	// Output: Sum64String("")    = 0xef46db3751d8e999

	h2 := xxhash.Sum64String("a")
	fmt.Printf("Sum64String(\"a\")   = 0x%x\n", h2)
	// Output: Sum64String("a")   = 0xd24ec4f1a98c6e5b

	// Hash a byte slice.
	h3 := xxhash.Sum64([]byte("asdf"))
	fmt.Printf("Sum64(\"asdf\")      = 0x%x\n", h3)
	// Output: Sum64("asdf")      = 0x415872f599cea71e

	// --- Streaming hashing ---

	// Create a new Digest and feed data in multiple writes.
	d := xxhash.New()
	d.WriteString("hello")
	d.WriteString(", ")
	d.Write([]byte("world"))
	fmt.Printf("streaming result  = 0x%x\n", d.Sum64())

	// Reset and reuse the same Digest.
	d.Reset()
	d.WriteString("hello, world")
	fmt.Printf("one-shot result   = 0x%x\n", d.Sum64())
	// Both lines above print the same value.
}
```

Expected output:

```text
Sum64String("")    = 0xef46db3751d8e999
Sum64String("a")   = 0xd24ec4f1a98c6e5b
Sum64("asdf")      = 0x415872f599cea71e
streaming result  = 0x<hash of "hello, world">
one-shot result   = 0x<hash of "hello, world">
```

<Tip>
  Prefer `Sum64String(s)` over `Sum64([]byte(s))` when you already have a string. Converting a string to `[]byte` allocates on the heap. `Sum64String` reads the string's memory directly, so it is both faster and allocation-free.
</Tip>

## Step-by-step walkthrough

<Steps>
  <Step title="Import the package">
    Add the v2 import path to your file:

    ```go
    import "github.com/cespare/xxhash/v2"
    ```

    All exported symbols are under the `xxhash` package name.
  </Step>

  <Step title="Hash a string in one call">
    Use `Sum64String` to hash a string and get a `uint64` result immediately:

    ```go
    h := xxhash.Sum64String("hello")
    fmt.Printf("0x%x\n", h)
    ```

    Use `Sum64` when your data is already a `[]byte`:

    ```go
    h := xxhash.Sum64([]byte{0x68, 0x65, 0x6c, 0x6c, 0x6f})
    fmt.Printf("0x%x\n", h)
    ```
  </Step>

  <Step title="Hash data in multiple chunks with Digest">
    Call `xxhash.New()` to create a `*Digest`, write your data in as many calls as you like, then read the final hash with `Sum64`:

    ```go
    d := xxhash.New()
    d.WriteString("part one")
    d.Write([]byte(" part two"))
    fmt.Printf("0x%x\n", d.Sum64())
    ```

    `Write` and `WriteString` both satisfy the `io.Writer` / `io.StringWriter` contracts and always return `(len(input), nil)`.
  </Step>

  <Step title="Reuse a Digest with Reset">
    Instead of allocating a new `Digest` for every hash, call `Reset` to clear the state and start over:

    ```go
    d := xxhash.New()

    d.WriteString("first input")
    h1 := d.Sum64()

    d.Reset()
    d.WriteString("second input")
    h2 := d.Sum64()

    fmt.Printf("h1=0x%x  h2=0x%x\n", h1, h2)
    ```

    `Reset` uses a zero seed. If you need a non-zero seed, use `ResetWithSeed(seed uint64)` instead.
  </Step>
</Steps>
