xxhash is a Go implementation of the 64-bit [xxHash](https://xxhash.com/) algorithm, XXH64. It is significantly faster than any hashing algorithm in the Go standard library, with throughput up to 17 GB/s on amd64 hardware.

## Why xxhash?

xxhash is designed for speed-critical applications where hash quality still matters — caches, hash maps, checksums, and deduplication pipelines.

<CardGroup cols={2}>
  <Card title="Up to 17 GB/s throughput" icon="bolt">
    Assembly-optimized paths for amd64 and arm64 achieve near-memory-bandwidth hashing speed.
  </Card>
  <Card title="Zero allocations" icon="leaf">
    Both one-shot (`Sum64`) and streaming (`Digest`) APIs are designed to produce zero heap allocations.
  </Card>
  <Card title="Implements hash.Hash64" icon="plug">
    Drop-in compatible with any code that accepts Go's standard `hash.Hash64` interface.
  </Card>
  <Card title="Seeded hashing" icon="key">
    Create independent hash spaces with `NewWithSeed` for applications like consistent hashing.
  </Card>

## Quick example

```go
package main

import (
    "fmt"
    "github.com/cespare/xxhash/v2"
)

func main() {
    // One-shot hashing
    h := xxhash.Sum64String("hello world")
    fmt.Printf("%016x\n", h)

    // Streaming hashing
    d := xxhash.New()
    d.WriteString("hello ")
    d.WriteString("world")
    fmt.Printf("%016x\n", d.Sum64())
}
```
